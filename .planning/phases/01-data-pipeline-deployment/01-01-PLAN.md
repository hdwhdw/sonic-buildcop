**`parse_gitmodules(content: str) -> list[dict]`:**
```python
parser = configparser.ConfigParser()
parser.read_string(content)
submodules = []
for section in parser.sections():
    # Section format: 'submodule "name"'
    name = section.replace('submodule ', '').strip('"')
    url = parser.get(section, 'url', fallback='')
    path = parser.get(section, 'path', fallback='')
    branch = parser.get(section, 'branch', fallback=None)
    # CRITICAL: use .removesuffix() not .rstrip() — rstrip('.git') mangles 'sonic-gnmi' to 'sonic-gnm'
    clean_url = url.removesuffix('.git').rstrip('/')
    parts = clean_url.split('/')
    if len(parts) >= 2:
        repo_slug = parts[-1]
        owner = parts[-2]
    else:
        continue
    if repo_slug in TARGET_SUBMODULES:
        submodules.append({
            "name": repo_slug, "path": path, "url": clean_url,
            "owner": owner, "repo": repo_slug, "branch": branch,
        })
return submodules
```

**`get_pinned_sha(session, submodule_path) -> str`:**
- URL: `f"https://api.github.com/repos/sonic-net/sonic-buildimage/contents/{submodule_path}"`
- Call `session.get(url)`, `resp.raise_for_status()`, `data = resp.json()`
- Verify `data["type"] == "submodule"`, raise `ValueError` if not
- Return `data["sha"]`

**`get_default_branch(session, owner, repo) -> str`:**
- URL: `f"https://api.github.com/repos/{owner}/{repo}"`
- Call `session.get(url)`, `resp.raise_for_status()`, return `resp.json()["default_branch"]`

**`get_staleness(session, owner, repo, pinned_sha, branch) -> dict`:**
- URL: `f"https://api.github.com/repos/{owner}/{repo}/compare/{pinned_sha}...{branch}"`
- Extract `commits_behind = data["ahead_by"]`
- If `commits_behind == 0`: return `{"commits_behind": 0, "days_behind": 0}`
- Otherwise: extract `pinned_date` from `data["base_commit"]["commit"]["committer"]["date"]` using `datetime.fromisoformat(date_str.replace("Z", "+00:00"))`
- Fetch HEAD date: GET `f"https://api.github.com/repos/{owner}/{repo}/commits/{branch}"`, extract `["commit"]["committer"]["date"]` same way
- Compute `days_behind = (head_date - pinned_date).days`, return `max(0, days_behind)`

**`build_compare_url(owner, repo, pinned_sha, branch) -> str`:**
- Return `f"https://github.com/{owner}/{repo}/compare/{pinned_sha}...{branch}"`

**`collect_submodule(session, submodule, retries=3) -> dict`:**
- For `attempt in range(retries)`:
  - Try: get_pinned_sha, resolve branch (submodule["branch"] or get_default_branch), get_staleness, build_compare_url
  - Return success dict: `{"name": submodule["name"], "path": submodule["path"], "url": submodule["url"], "pinned_sha": sha, "branch": branch, "commits_behind": staleness["commits_behind"], "days_behind": staleness["days_behind"], "compare_url": compare_url, "status": "ok", "error": None}`
  - Except `requests.RequestException as e`: if `attempt < retries - 1`: `time.sleep(2 ** attempt)`, continue
  - After loop exhaustion: return unavailable dict with all data fields as None, `"status": "unavailable"`, `"error": str(e)`

**`main()`:**
- `token = os.environ.get("GITHUB_TOKEN", "")`
- Create `session = requests.Session()` with `session.headers.update({"Authorization": f"token {token}", "Accept": "application/vnd.github.v3+json"})`
- Fetch .gitmodules: GET `https://api.github.com/repos/sonic-net/sonic-buildimage/contents/.gitmodules`, base64 decode content field: `import base64; content = base64.b64decode(data["content"]).decode("utf-8")`
- Parse: `submodule_defs = parse_gitmodules(content)`
- Collect each: `results = []` then for each submodule_def, call `collect_submodule(session, submodule_def)`, append to results, `time.sleep(0.5)` between calls
- Write output: `output = {"generated_at": datetime.now(timezone.utc).isoformat(), "submodules": results}`
- `with open("data.json", "w") as f: json.dump(output, f, indent=2)`
- Print summary: `ok_count`, `fail_count`, total
