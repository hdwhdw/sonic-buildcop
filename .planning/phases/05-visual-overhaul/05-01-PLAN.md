<objective>
Linkify submodule names and pinned SHAs as GitHub links, remove the redundant Path column,
add a header with project title/description, and add a footer linking to the source repo.

Purpose: Every entity on the dashboard becomes a direct link to its GitHub source — users click
through to repos and commits without copy-pasting. The Path column (always `src/<name>`) adds
noise now that the name itself links to the repo.

Output: Updated dashboard.html template (8 columns, links, header, footer) + updated tests
</objective>
