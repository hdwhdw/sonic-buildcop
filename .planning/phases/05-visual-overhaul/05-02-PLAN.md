<task type="auto">
  <name>Task 1: CSS dark mode, badge refinement, and professional polish</name>
  <files>templates/dashboard.html</files>
  <action>
Modify templates/dashboard.html — CSS changes in `<style>` block and badge Jinja markup:

**1. Badge text refinement (VIS-03):**

Change the badge template markup from:
```html
<span class="badge badge-{{ sub.staleness_status }}">{{ sub.staleness_status }}</span>
```
to:
```html
<span class="badge badge-{{ sub.staleness_status }}">● {{ sub.staleness_status | capitalize }}</span>
```

Leave the unknown badge as-is: `<span class="badge badge-unknown">—</span>`

In the CSS `.badge` rule, remove `text-transform: uppercase;` (the capitalize filter handles casing now).

**2. CSS refinements (VIS-01):**

Update these existing rules for tighter, more professional spacing:

- `.badge`: Change `border-radius: 4px` to `border-radius: 12px` (pill shape), change `padding: 2px 8px` to `padding: 2px 10px`
- `header` (new selector — add if Plan 05-01 didn't add styling): `margin-bottom: 20px; padding-bottom: 16px; border-bottom: 1px solid #e1e4e8;`
- `h1`: Change `margin-bottom: 4px` to `margin-bottom: 2px`
- `th, td`: Keep `padding: 8px 12px` (already good), add `vertical-align: middle` (if not present)
- `table`: Add `border: 1px solid #e1e4e8; border-radius: 6px; overflow: hidden;` for a contained table look
- `.summary`: Change `margin-bottom: 20px` to `margin-bottom: 0` (header border provides spacing)

**3. Dark mode (VIS-02):**

Add a `@media (prefers-color-scheme: dark)` block at the END of the `<style>` section (after all light-mode rules). Define dark variants for every colored element:

```css
@media (prefers-color-scheme: dark) {
    body {
        background: #0d1117;
        color: #c9d1d9;
    }
    a {
        color: #58a6ff;
    }
    th {
        background: #161b22;
    }
    th, td {
        border-bottom-color: #30363d;
    }
    table {
        border-color: #30363d;
    }
    tbody tr:hover {
        background: #161b22;
    }
    code {
        background: #21262d;
        color: #c9d1d9;
    }
    header {
        border-bottom-color: #30363d;
    }
    footer {
        border-top-color: #30363d;
        color: #8b949e;
    }
    .timestamp, .description {
        color: #8b949e;
    }
    /* Badge dark mode — slightly muted backgrounds for less eye strain */
    .badge-green { background-color: #238636; color: #fff; }
    .badge-yellow { background-color: #d29922; color: #fff; }
    .badge-red { background-color: #da3633; color: #fff; }
    .badge-unknown { background-color: #484f58; color: #c9d1d9; }
}
```

Note: Dark badge colors are GitHub's dark theme palette. Yellow badge changes from dark text (#333) to white (#fff) for dark mode readability.

**4. Also add `<html lang="en">` dark meta tag** — no change needed, the CSS media query handles it automatically via OS preference. Do NOT add any JavaScript for theme toggling.
