    **Constants:**

    MIN_COMMITS_FOR_CADENCE = 5
    MIN_MEDIAN_DAYS = 1.0
    LOOKBACK_DAYS = 180
    MAX_PAGES = 10

    FALLBACK_THRESHOLDS = {
        "yellow_days": 30,

    **Imports:**
    ```python
    import statistics
    from datetime import datetime, timezone, timedelta
    import requests
    import time
    ```

    **Function 1: get_commit_dates(session, owner, repo, branch, since_days=180)**
    - Compute `since` as `datetime.now(timezone.utc) - timedelta(days=since_days)`
    - Format as ISO string: `since.strftime("%Y-%m-%dT%H:%M:%SZ")`
    - GET `https://api.github.com/repos/{owner}/{repo}/commits` with params `sha=branch, since=since_str, per_page=100`
    - Loop: parse each commit's `commit.committer.date` into datetime, append to list
    - Follow pagination: `url = resp.links.get("next", {}).get("url")`, set `params = None` after first request
    - Safety cap: break after MAX_PAGES iterations
    - Sort dates ascending before returning
    - Add 0.5s courtesy delay between pages: `time.sleep(0.5)`

    **Function 2: compute_cadence(commit_dates: list[datetime]) -> dict**
    - If `len(commit_dates) < MIN_COMMITS_FOR_CADENCE`: return `{"median_days": None, "commit_count": len(commit_dates), "is_fallback": True}`
    - Compute intervals: `[(commit_dates[i+1] - commit_dates[i]).total_seconds() / 86400 for i in range(len(commit_dates) - 1)]`
    - Compute median: `max(statistics.median(intervals), MIN_MEDIAN_DAYS)`
    - Return `{"median_days": median, "commit_count": len(commit_dates), "is_fallback": False}`

    **Function 3: compute_thresholds(cadence: dict) -> dict**
    - If `cadence["is_fallback"]`: return a copy of `FALLBACK_THRESHOLDS`
    - Otherwise: `m = cadence["median_days"]`
    - Return `{"yellow_days": round(2 * m, 1), "red_days": round(4 * m, 1), "yellow_commits": 2, "red_commits": 4, "is_fallback": False}`

    **Function 4: classify(days_behind: int, commits_behind: int, thresholds: dict) -> str**
    - Helper `_level(value, yellow, red)`: return 2 if value > red, 1 if value > yellow, else 0
    - `day_level = _level(days_behind, thresholds["yellow_days"], thresholds["red_days"])`
    - `commit_level = _level(commits_behind, thresholds["yellow_commits"], thresholds["red_commits"])`
    - `worst = max(day_level, commit_level)`
    - Return `["green", "yellow", "red"][worst]`

    **Function 5: enrich_with_staleness(session: requests.Session, submodules: list[dict]) -> None**
    - For each submodule in submodules:
      - If `sub["status"] != "ok"`: set `staleness_status=None, median_days=None, commit_count_6m=None, thresholds=None`, continue
      - Call `get_commit_dates(session, sub["owner"], sub["repo"], sub["branch"])`
      - Call `compute_cadence(dates)`
      - Call `compute_thresholds(cadence)`
      - Call `classify(sub["days_behind"], sub["commits_behind"], thresholds)`
      - Set fields on sub dict: `staleness_status`, `median_days` (from cadence), `commit_count_6m` (from cadence["commit_count"]), `thresholds` (the thresholds dict)
      - Wrap in try/except for (requests.RequestException, KeyError, ValueError) — on failure, set all staleness fields to None and continue (defensive processing)
      - Add `time.sleep(0.5)` between submodules for rate-limit courtesy
